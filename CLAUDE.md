# IBK캐피탈 창립 40주년 기념 사이트

## 프로젝트 개요
- 배포 주소: https://gongmo-ibkc.github.io/ibk-capital-40th/
- 저장소: https://github.com/gongmo-ibkc/ibk-capital-40th (GitHub Pages, main 브랜치 루트)
- 단일 파일 사이트: `index.html` 하나에 CSS/JS/오디오(base64 2.8MB)까지 전부 내장
- 콘셉트: "혁신의 파도" — 40주년 헌정곡과 파도 모티프를 사이트 전체의 디자인 언어로 사용
- 이 프로젝트는 Claude(Cowork)에서 개발되어 Claude Code로 이관됨

## 파일 구조 (이 키트)
- `src.html` — **편집용 소스** (최신). 오디오 자리에 `__SAGA_SRC__`, 로고 자리에 `__LOGO_SRC__` 플레이스홀더. 모든 수정은 이 파일에서.
- `saga.mp3` — 40주년 헌정곡 「혁신의 파도」 (128kbps, 2:13.8)
- `logo.png` — IBK캐피탈 공식 로고 (500×112 투명 PNG, 14KB). 푸터(밝은 배경)에 원본 컬러로 사용.
- `logo-white.png` — 헤더용 리버스 버전. 심볼은 원본 컬러 유지, 글자만 흰색
  (logo.png에서 x≥128 영역의 불투명 픽셀을 흰색으로 치환해 생성. 전체 흰색 필터를 쓰면 심볼 디테일이 사라지므로 금지).
- `build.py` — src.html + saga.mp3 + logo.png + logo-white.png → index.html 생성
  (플레이스홀더 3개 치환: `__SAGA_SRC__`, `__LOGO_SRC__`, `__LOGO_W_SRC__`)
- `og-image.png` — 공유 카드 (1200x630, 저장소에 함께 배포)
- `index.html` — 빌드 결과물 (직접 편집 금지)

## 빌드 & 배포
```bash
python3 build.py          # index.html 생성
git add -A && git commit -m "..." && git push   # push → Pages 1~2분 내 자동 배포
```

## 사이트 구조 (풀페이지 스크롤 스냅, 15페이지)
1. 히어로 — 파도 캔버스(사가 재생 시 오디오 반응) + D-day 카운트다운(2026.11.1) + 40주년 엠블럼
2. 숫자로 보는 40년 — 통계 타일 8종(총자산·순이익·자기자본·임직원·ROA·자기자본비율 등) + 기록 배지
3~7. 성장 그래프 5페이지 — 총자산/당기순이익/자기자본/임직원/영업수익, 페이지 진입 시 드로잉 애니메이션
8~11. 40년의 항해 — 스토리텔링 연혁 4챕터(첫 물결→폭풍을 지나→너른 바다로→혁신의 파도), 시대별 배경 일러스트
12. 사업영역 (기업금융/투자금융/할부리스/글로벌)
13. 임직원 축하 한마디 — **Firebase Firestore 실시간 연동** (아래 상세)
14. 비전 + 푸터
- 오른쪽 도트 내비게이션, 헤더에 사가 재생 버튼

## Firebase (임직원 축하 한마디) — 구축 완료, 동작 검증됨
- 프로젝트: `ibk-40th` (Spark 무료), Firestore 서울(asia-northeast3), 프로덕션 모드
- 콘솔: https://console.firebase.google.com/project/ibk-40th/firestore
- 컬렉션 `cheers`, 문서: {name: string(≤20), msg: string(≤160), ts: serverTimestamp}
- 게시된 보안 규칙: 읽기 전체 허용 / create만 허용(필드·길이 검증) / update·delete 차단
- 클라이언트: src.html 하단 `<script type="module">` — gstatic CDN v10.12.2, onSnapshot 실시간 구독(최신 100개),
  addDoc 제출, localStorage 기반 1분당 1회 제한, textContent 렌더링(XSS 안전)
- firebaseConfig는 src.html에 내장돼 있음 (apiKey는 공개용 값이라 커밋해도 안전)
- 메시지 삭제(운영): Firebase 콘솔 → Firestore → cheers 컬렉션에서 문서 삭제 → 전 방문자 실시간 반영
- 현재 DB에 테스트 메시지 1건 있음("40주년 축하합니다! (테스트)") — 사용자가 콘솔에서 삭제 예정

## 주요 기능 구현 메모 (버그 이력 포함 — 재발 방지)
- **사가 자동재생**: 로드 시 play() 시도, 차단되면 첫 사용자 이벤트에서 재시도.
  ⚠️ 리스너는 재생이 "성공할 때까지" 유지해야 함. 휠/스크롤은 브라우저가 제스처로 인정하지 않아
  play()가 거부되는데, 과거에 첫 이벤트에서 리스너를 제거해버려 음악이 영영 시작 안 되는 버그가 있었음.
- **오디오 비주얼라이저(Web Audio)**: AudioContext는 반드시 진짜 제스처(pointerdown/keydown/touchstart)
  문맥에서만 생성. suspended 상태에서 createMediaElementSource를 연결하면 소리가 사라지는 무음 버그 있음.
- **가사 자막(CUES 배열)**: 사용자가 LRC 메이커(lrc-maker.github.io)로 직접 찍은 실측 타임코드(0.1초 단위).
  임의 수정 금지. 첫 가사 14.06s. 1:15~1:27 사이 12초 간주 구간에 "♪ ── 간주 ──" 표시 로직 있음.
  가사 박스는 20px(모바일 15px) — 사용자가 1.4배 확대 요청했던 결과값.
- **차트(renderChart)**: 선그래프는 y축을 데이터 범위로 확대(0 시작 아님, 각주로 명시), 막대는 항상 0 기준
  (길이 왜곡 방지). 로그 스케일은 구현했다가 사용자 요청으로 제거함. 드로잉 애니메이션은 .chart-in 클래스
  (IntersectionObserver, 페이지 재진입 시 재실행).
- **풀페이지 스냅**: html{scroll-snap-type:y mandatory}, 모바일/저높이 화면에서는 proximity로 완화.
- **폰트**: Pretendard(jsdelivr CDN), 실패 시 시스템 폰트 폴백.

## 데이터 출처 (전부 공시·공식 자료로 검증됨 — 임의 수정 금지)
- 총자산·순이익·자기자본 2010~2017: 기업은행 영문 연결감사보고서 자회사 요약재무 (global.ibk.co.kr)
- 2019~2021 순이익: 기업은행 애뉴얼리포트 (별도 기준)
- 2021~2025 총자산·순이익·ROA·자기자본비율: IBK캐피탈 주요경영지표 (ibkc.co.kr)
- 2023~2025 자기자본·영업수익: IBK캐피탈 재무현황 (별도 기준)
- 임직원 수 2010~2025: 금융감독원 FISIS 임직원현황 (각 연도 12월 말 총임직원, 임원 포함)
- 의도적 공백: 2018년 연간 순이익, 2019년 말 총자산(공개자료 미확인 — 그래프에서 비워둠, 각주 표기)
- 연혁: IBK캐피탈 공식 주요연혁 페이지 기준

## 개발 히스토리 요약 (Cowork 세션에서 진행된 순서)
1. Canva 커넥터 연동 테스트 → 40주년 포스터 시안 4종 생성 (참고용)
2. 홈페이지 초안: 히어로/통계/연혁/사업영역/비전 (단일 HTML)
3. 성장 그래프 추가 → 2010년대 데이터 발굴(기업은행 감사보고서·FISIS)로 15개년 시계열 확장
4. 사가 MP3 내장 + 재생 버튼 → 자동재생 + 버튼 문구 "40주년 헌정곡 - 혁신의 파도"
5. GitHub Pages 배포 (저장소 gongmo-ibkc/ibk-capital-40th, 웹 업로드 방식)
6. 임직원 수 그래프 추가 (FISIS를 브라우저로 직접 조회해 수집)
7. 파도 콘셉트 리디자인 + 4챕터 스토리텔링 연혁
8. 풀페이지 스크롤 전환, 통계 8타일 확장, 그래프별 1페이지
9. y축 로그 토글 추가 → 제거하고 데이터 범위 확대 방식으로 변경
10. 고도화 7종: 차트 드로잉 애니메이션/엠블럼/OG 카드/시대별 일러스트/가사 자막/D-day/모바일 점검
11. 가사 싱크: 에너지 분석 → 사용자 실측 LRC로 최종 확정
12. 축하 한마디: Google Sheets 연동 → Firebase Firestore 실시간으로 전환 (콘솔 설정 포함 완료)
